Disaster Awareness API

Welcome to the Disaster Awareness API. Here are the available endpoints:

Authentication

POST /api/login - User login

POST /api/register - User registration

Kits

GET /api/kit - Get all kits

POST /api/kit - Create new kit

GET /api/kit/{id} - Get kit by ID

PUT /api/kit/{id} - Update kit

DELETE /api/kit/{id} - Delete kit

Content

GET /api/content - Get all content

POST /api/content - Create new content

To test the API, you can use tools like Postman or curl. For example:

curl -X POST http://localhost:8080/disaster-awareness/api/login \
  -H "Content-Type: application/json" \
  -d '{"email":"test@example.com","password":"password123"}'
